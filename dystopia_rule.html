①難易度変更提案＆投票について
[概要]
本難易度表で搭載中の差分に関する、難易度変更提案及び投票を行えます。
24時間365日いつでも変更提案が可能。

[導入]
どなたでも参加可能です。
最初期に行った投票システムを見直し、もう少し手軽かつ簡単に提案や投票することが可能(？)なシステムに変更しました。
※試験的に運用のため至らない点があるかもしれません、ご承知おきください。

提案フォーム及び投票フォームには、PLAYER IDとPLAYER NAMEの入力が必須になります。
提案状況は常時公開とするためID及びNAMEが公開されます、ご容赦くださいませ。

提案＆投票の前には、本ページ内容に目を通していただくようお願いいたします。

[提案について]
いつでも好きな時に提案フォームから難易度昇格、降格提案及び削除、隔離提案(dy？行き)が可能です。
dy0より弱いまたはdy7より強いと思われる譜面に関しては、削除提案や隔離提案ではなく降格提案もしくは昇格提案で提案をお願いいたします。
投票結果が開示されたものに関しては、すぐにまた提案を出すことが可能です。

提案フォームは下記5つの項目に分かれてます。
1回答につき1曲とし、それぞれの項目に従った回答をお願いします。

・PLAYER ID
LR2IR IDもしくはMinIR IDを記載してください。
※IDのみ記載してください。

・PLAYER NAME
Stella難易度表様、Twitter、IRでの名前等自由に記載ください。

・現在搭載中の難易度
変更提案を行った差分が現在搭載されている難易度を選択してください。

・曲名
平衡世界 [miracle]
上記のように差分名まで記載ください。
IR等からのコピー＆ペーストを推奨します。

・希望提案
昇格希望提案、降格希望提案、削除希望提案、隔離希望提案の4つから1つお選びください。

[投票について]
提案者が提案した案件に対して、賛成(100%)、中立(50%)、反対(0%)の3つから投票が可能です。
この時、提案者は自身で提案した案件に投票を行えません(投票開始時に管理人が提案者賛成票を1票入れます)。

・dy0-4について
昇格及び降格提案は提案者を含めて2票以上の賛成票を必要とし、65%以上で可決となります。
※削除提案は、提案者を含めて最低3人以上の参加者と2票以上の賛成票、65%以上を必要とします。

・dy5-7について
昇格及び降格提案は提案者を含めて1票以上の賛成票を必要とし(提案者のみでも有効)、65%以上で可決となります。
※削除提案は、提案者を含めて最低2人以上の参加者と2票以上の賛成票、65%以上を必要とします。

・隔離票について
隔離提案は、提案者を含めて3人以上の投票参加、65%以上で可決となります。
管理人的にはdy？の曲数をあまり増やしたくないので、｢難易度バランスが全体的に取れているけど難易度がつけにくい｣差分以外は削除提案でお願いします。

投票フォームは下記最低4つの項目に分かれてます。
それぞれの項目に従った回答をお願いします。

・PLAYER ID
LR2IR IDもしくはMinIR IDを記載してください。
※IDのみ記載してください。

・PLAYER NAME
Stella難易度表様、Twitter、IRでの名前等自由に記載ください。

・現在搭載中の難易度
変更提案を行った差分が現在搭載されている難易度を選択してください。

・曲名A、○○提案について
賛成、中立、反対の3つから1つお選びください。
※○○には昇格、降格、削除、隔離のいずれかの言葉が入ります。

・曲名B、○○提案について
…(略)

以降同様にして、提案があった数だけ項目数が追加されます。
※投票フォームはあらゆる提案を含みます、それぞれ任意回答としますので必要な項目のみ回答してください。

[投票期間について(重要)]
提案はいつでも可能ですが、投票用フォームで管理する都合上集計と投票の期間を定めます。

変更提案の意見を出揃える→投票期間
という流れで行います。

それぞれの日程は下記の通りです。
変更提案 : いつでも可能
提案集計日 : 毎月第2週の金曜日
(集計期間 : 前回の提案集計日から今回の提案集計日前日まで)
投票開始日 : 提案集計日の翌日(毎月第2週の土曜日)
(投票可能期間 : 13日間)

提案集計日までに出揃った提案をまとめ、管理人が投票用フォームを作成します。
投票開始期間になりましたら、投票用フォームへのリンクを掲載いたしますのでそちらから投票への参加をお願いします。

例、今日が2022年5月13日(金)の場合
4/8(金)〜5/12(木)までの期間で提案された提案について、管理人が投票用フォームを作成します。
翌日5/14(土)に投票用フォームが公開され、公開日を含めて13日間の投票期間が開始されます。
13日後の5/26(木)が終わる5/27(金)の0:00に投票が締め切られ、集計結果が第4週の金曜日である5/27(金)中に発表されます。

[制限、その他]
提案は、集計日間で1人あたり7回までとさせていただきます。
※やむを得ず、管理人はこの制限回数を越えて提案を行う場合があります。
誹謗中傷等、その他無関係な書き込みなどが発見された場合は、適切な対応を取らせていただく場合があります。




②新規搭載リクエストについて
[概要]
どなたでもDystopia難易度表への差分搭載リクエストを行うことができます。

[導入]
LR2で正常に動作しないものに関してはリクエストを受け付けません。

自作譜面の自薦を可とします。
1人あたり制限無く何回でも回答可能です。
回答に対してペナルティ等はありませんが、誹謗中傷等が確認された場合は適切な対応を取らせていただくことがあります。

本難易度表への提案要件を満たすと思われるものについて回答ください。
非搭載要件を全て満たさないものが、提案要件を満たすものとします。
非搭載要件は次の通りです。

[非搭載要件]
・既にStella正式難易度表様に搭載されているもの(回答時点で搭載されていないものは問題ないです)。
・極端な皿やLN、とても長い棒のような縦連や壁、ソフランやギミックなどの個人差もしくは対策、覚え要素等が多く、EASY CLEARまたはHARD CLEARに大きく影響する。
・繰り返し同じ配置が極端に多い、または非常に長い。
・ダミーノート(地雷オブジェ)を使用している。 ・暇な時間が長いか、道中と殺し地帯の難易度差が極端に離れている。
・極端に低いTOTAL値または高いTOTAL値である。
・演奏時間が極端に長い(3分半〜4分以上)。
・演奏時間が30秒以下(回答は可能としますが、搭載は検討します)。
・地力譜面と言い難い(難易度非適正プレイヤーが数段階難易度を飛ばしてEASY CLEARできてしまう等)。
・版権曲や著作物、他人を不快にさせるもしくは中傷したり、犯罪を助長するようなもの等。
・楽曲作成者または差分作成者が難易度表への搭載を拒絶している。
・本体または差分のURLが壊れており、現在ではDL不可能なもの(補完して回答するのは可能とします)。

[搭載について]
回答いただいたものは、検討を経て搭載いたします。
搭載日は通常の更新と同じになりますが、管理人の精査速度次第では遅くなる場合もあります。
非搭載要件に1つでも当てはまると思われるものは非搭載となります。
管理人目線でギリギリのラインのものは検討次第で非搭載となります、必ずしも搭載されるとは限りませんのでご容赦くださいませ。
また、搭載時・非搭載時の回答者へのフィードバックはありません。

初期搭載時は譜面数が乏しかったのを補うために数多の譜面を採用しましたため、非搭載要件を抱えて搭載されているものもあるかもしれません。
ですが、今後譜面数が増えていくに従ってそういったもの等は削り(アンケート式を予定)をいれていく予定ですのでご理解いただけると幸いです。

[回答方法]
回答欄は下記7つの項目に分かれてます。
1回答につき1曲とし、それぞれの項目に従った回答をお願いします。

・PLAYER ID
LR2IR IDもしくはMinIR IDを記載してください。
※IDのみ記載してください。

・PLAYER NAME
Stella難易度表様、Twitter、IRでの名前等自由に記載ください。

・曲名
平衡世界 [miracle]
上記のように差分名まで記載ください。
IR等からのコピー＆ペーストを推奨します。

・bmsmd5
md5とは各譜面が固有に持つハッシュ値のことで、同曲同差分名のものと区別するために重要な項目となります。
BMS Score Viewer様にて該当の譜面ファイルをD＆Dもしくは検索にて開き、URLの｢www.ribbit.xyz〜(中略)〜md5=xxx…｣より"xxx…"以降全てをコピーして回答欄へペーストをお願いします。

・本体URL
イベントURLを可能とします。
URLが壊れていないことを確認してください。
正規URLが壊れている場合は補完して回答いただいても構いません。

・差分URL
本体の中に初めから同梱されている譜面の場合は｢同梱譜面｣とご記載ください(カギカッコは不要)。

・推定難易度
dy0-7から提案差分の推定される難易度をお選びください。
dy？への回答は受け付けておりません。
