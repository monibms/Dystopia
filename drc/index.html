DRCについて

概要

DRC (Dystopia Rank Certification : 段位認定)
プレイヤーのおおよその実力を測るためのコンテンツです。
ページ下部にDRC Set Listを記載してあります(下に行くほど難易度が高くなります)。

コンセプト

さまざまな傾向の地力が一定程度あれば受かるような、上から殴るようなタイプの段位を目指しました(EASYでそこそこ遊べる難易度-1のコースが適正目安を想定)。
回復枠や極端な局所難を含む譜面はほとんど採用していないつもりです。
ゲージ推移は常に横ばいで瀕死を想定して作成したので、どの傾向にも限らず地力や体力が不足していると難しく感じると思います。
※上記のようなコンセプトがありますので、他の難易度表の段位との難易度比較は難しいと思います。

構成

DRCは当該表記難易度のみの楽曲/譜面で構成されています(作成時は当該難易度でも、以降の難易度変更によって難易度が異なってしまう場合もあります)。

合格者数履歴

DRCを受けるプレイヤーへの見返りが何も無いのは寂しかったので、軽めのモチベ向上も兼ねて毎月合格者の総数をログとして残していきます。
更新は月に1回で、毎月最初の楽曲更新日(第2週の金曜日)にLR2IR及びMinIRから合格者数をカウントします。
更新の際は、合格者数をカウントした時刻(JST)も合わせて表記します。
人類学における正確なカウントを行いたいですが、不正等を見抜くシステムは無いのでデータ改ざんやオートプレイ用プログラムの使用、連射パッド使用等の行為は避けていただけると幸いです。

おわりに

全ての段位コースを2人だけで作成したので不満な点もあるとは思いますが、至らない点が多いことご容赦いただければ幸いです。
この段位(DRC)が、たくさんのプレイヤーのモチベーションや地力の向上に繋がることを願っております。
